MACRO TO MICRO

UI/UX Exhibition design Interactive Game Microplastic

OVERVIEW

Macro to Micro is an interactive board game where players navigate a map using everyday objects as sources of microplastics (car tires, shampoo, clothing, plastic bottles).
Each step reveals the journey through animations and additional information. Designed as a multi-player experience, the game combines learning with competition through quiz-based interactions.

PROCESS

01 Research

We explored how microplastics enter the human body and identified key pathways. The goal was to break down a complex topic into clear, understandable steps and define the most relevant information for the experience.

02 Concept

Based on the research, we developed the idea of an interactive board game that makes these pathways tangible through play. Physical objects represent sources of microplastics, while a connected screen provides additional information, animations, and quiz elements.

03 Implementation

We structured the experience into a step-by-step journey. Players move across the board, triggering digital content that gradually explains each stage. The flow combines navigation, learning, and interaction in a clear and intuitive sequence.
